# FoodLoop
FoodLoop merupakan platform website yang mendukung upaya zero hunger serta pengurangan food waste melalui upaya redistribusi surplus pangan layak konsumsi kepada pihak membutuhkan sekaligus distribusi limbah makanan yang tidak layak konsumsi kepada petani dan peternak untuk diolah menjadi produk fungsional seperti pupuk organik dan pakan ternak
